# Contributing to Seven — 3-DOF Arduino Robotic Arm

Thank you for your interest in contributing! Here's how you can help.

## Ways to Contribute

- Report bugs or issues
- Suggest new features or improvements
- Improve documentation
- Share your own builds or modifications

## Reporting Bugs

1. Go to the [Issues](https://github.com/elevenMiles/Robotic_Arm_Seven/issues) tab
2. Click **New Issue**
3. Describe the problem clearly — include:
   - What you expected to happen
   - What actually happened
   - Your Arduino IDE version
   - Any error messages from the Serial Monitor

## Suggesting Features

Open an issue with the title starting with `[Feature Request]` and describe your idea.

## Submitting Code Changes

1. Fork the repository
2. Create a new branch: `git checkout -b my-feature`
3. Make your changes
4. Test thoroughly on hardware before submitting
5. Submit a Pull Request with a clear description of what you changed and why

## Code Style

- Follow the existing file structure (`roboticArm`, `servoConfig`, `vector3`, etc.)
- Keep constants in `config.h`
- Comment any non-obvious logic, especially in kinematics

## Questions?

Open an issue or reach out via GitHub.
